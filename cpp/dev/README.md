# DEV

A C++ development environment with building system `[Blade](https://github.com/chen3feng/blade-build)`.

Includes,

- boost 1.74
- python3
- gcc 8.3.0
- gdb 8.2.1
- clang 7.0.1
- lldb 7.0.1
- blade 2.0
- cmake 3.13.4
- cppcheck 1.86
- valgrind 3.14.0
- gperftools 2.9.1
- vim 8.1


# Download

You can download the built image with 

```
docker pull registry.cn-hangzhou.aliyuncs.com/jackeylv/cpp:dev
```
